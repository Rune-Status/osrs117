# OSRS #117
A RuneScape private server prototype which demonstrates the protocol usage of the #117 revision.

Currently supports:
* Handshaking (Duh)
* OnDemand decoding and streaming of cache files
* Login decoding
* Outgoing game packets
  * Player initialization
  * Region Change
  * Window pane
